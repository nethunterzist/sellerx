- Çoklu mağaza yönetimi

### Talebimiz

Satıcılarımızdan gelen yoğun talep doğrultusunda, **Trendyol Reklam (Advertising) verilerine programatik erişim** sağlayabilmek istiyoruz. Bu entegrasyon sayesinde satıcılarımız:

1. **Ürün Reklamları** - Kampanya performansını (gösterim, tıklama, harcama, ROI) tek panelden takip edebilecek
2. **Mağaza Reklamları** - Mağaza tanıtım kampanyalarının etkinliğini analiz edebilecek
3. **Influencer Reklamları** - Affiliate kampanya sonuçlarını görüntüleyebilecek
4. **Bütçe Yönetimi** - Reklam bütçe durumunu anlık izleyebilecek
